# NT_exercicios
Exercícios propostos em JAVA

:: Exercícios ::

1:. Mostre no console todos os números pares entre 0 e 10 mil
2:. Escreva um algoritmo que calcule o intervalo entre dois pontos no eixo x
3:. Escreva um algoritmo que recebe do usuário peso e altura e calcule o (IMC), índice de massa corpórea, do usuário

4:. O brinquedo de uma loja custa 8 reais e 50 centavos. Escreva um algoritmo que recebe um valor entre 0 e 100 reais e calcula o troco para o brinquedo em notas e moedas. O troco deve devolver o menor número de notas e moedas facilitando assim o troco para compras futuras

5:. Crie uma classe Usuário, crie também as classes Funcionario e Cliente, ambas sendo uma extensão de Usuário

	Usuário: (nome, idade, cpf)
	Funcionario: (salario, cargo)
Cliente: (saldo)

Crie 5 funcionários e 3 clientes, printe o nome dos funcionários e o cpf dos clientes

6:. Faça um algoritmo que remova todos os números ímpares de uma determinada lista de inteiros
Exemplo:
Lista: [2, 4, 8, 13, 17, 22, 15, 31]
Resultado:
Lista: [2, 4, 8, 22]

7:. Escreva um algoritmo que ordene uma lista qualquer em ordem crescente
Exemplo de lista para teste: [10, 30, 12, 7, 50, 500, 3]
